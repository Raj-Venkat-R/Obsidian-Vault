# What is an Array?

An **array** is a data structure that stores **multiple elements of the same data type** in a **contiguous block of memory**.

In simple words:
> An array keeps related data **side by side** in memory and lets us access each element using an **index**.

## ðŸ”¹ Contiguous Memory (MOST IMPORTANT IDEA)

**Contiguous memory** means:
- All elements are stored **next to each other**
- No gaps between elements
- Memory addresses increase sequentially

- Takes the **same amount of memory**
- Is placed immediately after the previous one

## ðŸ”¹ Why Contiguous Memory Matters

Because elements are stored side by side, the computer can:
- Calculate the location of any element directly
- Jump to it **without scanning** other elements

## ðŸ”¹ What Can an Array Store?

An array stores:
