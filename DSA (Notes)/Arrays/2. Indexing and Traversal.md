# Indexing & Traversal in Arrays

## ðŸ”¹ What is Indexing?

**Indexing** is the way we **identify and access** each element in an array.

- Every element is assigned a **number** called an **index**
- Index represents the **position** of the element
- Indexing starts from **0**

### Example

```
Array â†’  [ 10 , 20 , 30 , 40 , 50 ]
Index â†’    0    1    2    3    4
```

- First element â†’ index `0`
- Last element â†’ index `length - 1`

---

## ðŸ”¹ Why Indexing Starts from 0?

Because array indexing is based on **memory offset**.

If:
- Base address = starting memory
- Index = offset from base

Then:
- Index `0` â†’ base address
- Index `1` â†’ base + size of one element

This makes address calculation simple and fast.

## ðŸ”¹ What is Traversal?

**Traversal** means:
> Visiting **each element** of the array **one by one**

Traversal is required when:
- Printing elements
- Searching
- Finding max/min
- Calculating sum
- Applying logic to all elements

---

## ðŸ”¹ Traversal Using for-loop (Most Important)

```java
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

### How it Works (Diagram)

```
i = 0 â†’ arr[0]
i = 1 â†’ arr[1]
i = 2 â†’ arr[2]
...
```

Stops when:
```
i == arr.length
```

### Key Difference

- Use enhanced loop only when **index is not needed**.

### Reverse Traversal
